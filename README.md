## What is VEKA Network?

VEKA Network is a **multi-gamemode Minecraft network** built for the Indian Minecraft community and beyond. Choose from **8 different gamemodes** — all on one server with seamless world switching. Whether you want to survival build, grind prison ranks, wage faction wars, or dominate bedwars — there's something here for you.

**No pay-to-win. No clutter. Just gameplay.**

## How to Join

1. Open Minecraft **Java Edition** (version 1.8 — 1.21)
2. Click **Multiplayer** → **Add Server**
3. Enter the server address: **`mc.veka.gg`**
4. Click **Join Server** and enjoy!

> **Note:** This server supports both premium and non-premium accounts. Register with `/register <password> <password>` on first join. Login with `/login <password>` on subsequent joins.

## Server Rules

We keep rules simple. Breaking them results in warnings, mutes, or bans.

1. **No Griefing** — Don't destroy or steal from other players
2. **No Hacking** — No hacked clients, x-ray, or unfair mods (Grim enforces this)
3. **No Spam** — Don't flood chat or advertise
4. **Be Respectful** — Treat everyone with respect
5. **No Duping** — Don't exploit duplication glitches
6. **English in Chat** — So staff can moderate fairly
7. **No Lag Machines** — Don't build intentional lag machines
8. **Respect Claims** — Don't build in other players' claimed areas

## Frequently Asked Questions

**Q: Can I play with friends?**
A: Absolutely! Claim land in survival, form a faction, create a town, or just play together.

**Q: What version do I need?**
A: Minecraft Java Edition 1.8 — 1.21. ViaVersion is installed so all versions work.

A: We have no planned world resets. Your builds are safe.

A: Yes! Cracked mode is enabled. Register with `/register <password> <password>`.
